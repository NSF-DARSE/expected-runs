Expected Runs Documentation
===========================

Overview
--------
**Run expectancy** is expected runs a team score from a particular 
game state for the rest of the half inning. A game state is 
determined by the number of players in base, number of outs, number
of strikes, and number of balls. In general, the state of game can 
also depend on several factor including the batting line-up, pitcher, 
stadium, weather, etc.
